Please run Powershell as Administrator and go to file path which you keep M365GrMembExp.ps1 file.

Once you run the script as typing .\M365GrMembExp.ps1, it prompts you to log in to Exchange Online with your M365 Administrator account.

Then, it will list your Microsoft 365 groups with all their memberships. You can find all the exported data in the csv file which located under C:\ExportM365Folder.
